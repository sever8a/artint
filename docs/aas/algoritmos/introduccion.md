# Introducción

El aprendizaje automático supervisado utiliza un conjunto de datos de entrenamiento previamente etiquetados, donde se conoce el valor del atributo objetivo o respuesta (*Tag*). Esto permite que el algoritmo aprenda a generalizar y predecir respuestas para nuevos datos.

Los métodos supervisados se dividen en dos categorías principales:
- **Regresión**: para predecir valores continuos
- **Clasificación**: para predecir categorías

Tanto los árboles de decisión como las redes neuronales son alternativas potentes que pueden aplicarse tanto a problemas de regresión como de clasificación.

!!! alert "Consideraciones importantes"

    **Volumen de datos**: Las redes neuronales requieren miles de datos para obtener buenos resultados. Con pocos datos, otros modelos pueden ser más efectivos.

    **Estrategia inicial**: Comienza con modelos más simples antes de implementar redes neuronales complejas.

    **Interpretabilidad**: Las redes neuronales son difíciles de explicar. En entornos laborales, la capacidad de justificar predicciones es crítica, lo que limita su uso en algunos casos.

Comprender los fundamentos básicos es esencial para avanzar en este campo.
