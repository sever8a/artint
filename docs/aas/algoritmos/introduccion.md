# Introducción
El aprendizaje automático supervisado parte de un conjunto de datos de entrenamiento que están previamente etiquetados, en el cual se conoce el valor del atributo objetivo o respuesta (o *Tag*). 

Esto hace que el algoritmo pueda aprender y obtener una función que sea capaz de generalizar y predecir la respuesta para un conjunto de valores nuevo.

Existen dos grandes grupos de métodos supervisados: algoritmos de *regresión* y algoritmos de clasificación, árboles de decisión y redes neuronales. Lo pueden **hacer todo**.

!!! alert "Requisitos"

    Para que el modelo sea interesante plantearlo con redes neuronales, se deben tener miles de datos. Si no es así, no se consiguen buenos resultados con pocos datos.

    Utilizar otros modelos en un primer momento es una buena estratégia para comenzar a resolver un problema.

    Requiere más recursos. En el mundo laboral no son buenos resultados, ya que cualquier modelo de red neuronal resulta **poco explicable**. Su complejidad para interpretar los resultados, aconseja poco utilizarlas.


Basta con entender los fundamentos básicos para avanzar en el conocimiento.
