# Introducción

Los modelos de Machine Learning, tienen que ser entrenados para cada tarea concreta. El NLP permite generar modelos más generales.

Comienza en los años 50, pero son los recursos computacionales. 

En una primera fase se establecen modelos basados en reglas, de manera manual para validar los mensajes. Con las nuevas variaciones o significados interpretativos, el modelo no resulta suficiente.

!!! alert "Traductor Google"

    Hace unos años el funcionamiento era muy literal sin establecer un contexto para generar los textos.

Las máquinas aprenden representaciones del lenguaje y del mundo a partir de datos. Este avance ha convertido al NLP en la **principal interfaz de comunicación** entre humanos y máquinas.

!!! info "Sin competencia en IA"

    Problemas con la competencia profesional si no se conoce y utilizan las herramientas basadas en IA.

## NLP Procesamiento Lenguna Natural

* Entender y generar texto, superando la limitación de hacer solo treas específicas.
* Permite aprovechar modelos preentrenados que se pueden ajustar a diferentes tareas de lenguaje con menos datos.

## Beneficios clave

- **Multitarea** Un único modelo puede realizar varias tareas de lenguaje.
- **Escalabilidad y generalización** Mejor desempeño en tareas para las que no ha sido entrenada.
